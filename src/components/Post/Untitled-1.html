Challenge 9: Create a Module

Estimated Time: 10-15 minutes

Modules are used to organize course content by weeks, units, or whatever organizational structure works for your course. With modules, you are essentially creating a one-directional linear flow of what you would like your students to do. You can use Modules to:

  • create prerequisite activities that students must complete before moving on in the course
  • track student progress through a sequence of learning activities
  • organize course content by unit, day, week, topic or Outcome

Challenge Directions:

In this Challenge, create a Module to organize everything you've created. Add the following content to the Module:

  1. Lesson Plan Page
  2. Discussion
  3. Quiz
  4. Assignment

To get started click on the Modules tab in the Navigation Bar > Click the blue Create a Module button in the upper right hand corner of the screen > Give your Module a new title > Select the option to Lock the Module until a given date and or set a Prerequisites having students follow a series of tasks in sequential order > Finally, click Add Module.

To Add Content:

After you have created a Module you can add a variety of content by clicking the + button at the top of each Module. > Click the drop down menu to add some content. You can add a/an:

  • Assignment
  • Discussion
  • Quiz
  • Page
  • File

When you add a new item, it is important to fill out the empty fields that appear such as Page Name etc...

It is important to make sure to publish your Module and all of the contents in it be visible for students to access.

 

To Publish a Module:

To publish a Module click the little gray cloud icon next to each Module Title as well as any other items you wish for students to view. You will know when a Module or item within a Module is published once the little gray cloud turns green. Modules and items within a Module can be Published and Unpublished at any time.
